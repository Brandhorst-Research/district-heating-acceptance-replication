# district-heating-acceptance-replication
# Replication materials for: Psychological Determinants of Renewable District Heating Acceptance: A Theory of Planned Behavior Study

This repository contains the data and code to reproduce the analyses
reported in:

Leif Donar Brandhorst (2026). Psychological Determinants of Renewable District Heating Acceptance: A Theory of Planned Behavior Study.

## Contents
- data/raw: raw data
- code/: analysis scripts
- outputs/: tables and figures

## Reproduction

The full analysis is contained in `code/analysis.Rmd`.

A knitted Word report including all code, tables, and figures is available at:
- outputs/main_report.docx

To reproduce the report:
1. Open `code/analysis.Rmd` in RStudio.
2. Install required R packages.
3. Click "Knit to Word".

The knitted document will reproduce all results reported in the manuscript.
