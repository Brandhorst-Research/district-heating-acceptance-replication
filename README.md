# district-heating-acceptance-replication
# Replication materials for: Psychological Determinants of Renewable District Heating Acceptance: A Theory of Planned Behavior Study

This repository contains the data and code to reproduce the analyses reported in:

Leif Donar Brandhorst (2026). Psychological Determinants of Renewable District Heating Acceptance: A Theory of Planned Behavior Study.

## Contents
- code/ – R Markdown file containing the full analysis
- data/raw/ – anonymized raw survey data
- outputs/ – knitted Word report generated from the analysiss

## Reproduction

This repository represents the complete R project used for the analysis.

To reproduce the results:

1. Download the full repository.
2. Open `code/analysis.Rmd` in RStudio.
3. Install required R packages.
4. Knit the document to Word.

The knitted report reproduces all tables and figures reported in the manuscript.
